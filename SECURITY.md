# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in DBKeep, please report it by sending an email to **security@dbkeep.com**.

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact

We will respond within **48 hours** and work with you to understand and resolve the issue promptly.

**Please do not open a public GitHub issue for security vulnerabilities.**
